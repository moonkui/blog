Vue基础2

实例的生命周期

最主要的八个生命周期
beforeCreate、created、beforeMount、mounted、beforeUpdate、updated、beforeUnMount、unmounted(创建前、创建后、挂载前、挂载后、更新前、更新后、卸载前、卸载后)
其中beforeUpdate和updated可能不执行，也可能不只执行一次
官方文档提示：
不要在选项 property 或回调上使用箭头函数，比如 created: () => console.log(this.a) 或 vm.$watch(‘a’, newValue => this.myMethod())。因为箭头函数并没有 this，this 会作为变量一直向上级词法作用域查找，直至找到为止，经常导致 Uncaught TypeError: Cannot read property of undefined 或 Uncaught TypeError: this.myMethod is not a function 之类的错误。

文本语法

文本插值

数据绑定最常见的形式，两个大括号，当同名中的data里的数据发生改变，插值处的内容也会更新
相反，使用v-once能执行一次性插值，数据改变时不更新

v-html

双打括号会将数据输出为普通文本，v-html则是输出真正的HTML
官方文档提示：
在你的站点上动态渲染任意的 HTML 是非常危险的，因为它很容易导致 XSS 攻击。请只对可信内容使用 HTML 插值，绝不要将用户提供的内容作为插值。

Vue基础

声明式渲染

使用JavaScript表达式

对于所有的数据绑定，Vue.js都提供来完全的JavaScript表达式的支持
但是有个限制就是每个绑定都只能包含单个表达式

v-bind

绑定元素的属性
如果绑定的值是null或undefined，那该attribute将不会被包含在渲染的元素上
对于v-bind如果disabled的值是truthy，那么disabled attribute将包含在内，如果该值是一个空字符串，它也会包含在内，与

保持一致.对于其他错误的值，该attribute将被省略

Truthy(真值)

指的是在布尔值上下文中，转换后的值为真的值。所有值都是真值，除非它们被定义为假值（即除false、0、””、null、undefined和NaN以外的皆为真值）

v-on

添加事件监听器

除了直接绑定到一个方法，也可以在内联JavaScript语句中调用方法

事件修饰符

为了阻止默认事件，修饰符是由点开头的指令后缀来表示的
.stop 阻止单击事件继续传播
.prevent 提交事件不再重载
.capture 添加事件监听器时使用事件捕获事件（即内部元素触发的事件先在此处理，然后才交由内部元素进行处理）
.self 只当在event.target是当前元素自身是触发处理函数（即事件不是从内部元素触发的）
.once 点击事件只会触发一次
.passive
提示：修饰符可以串联，但顺序很重要；相应的代码会以同样的顺序产生。因此，用 v-on:click.prevent.self 会阻止所有的点击，而 v-on:click.self.prevent 只会阻止对元素自身的点击。

按键修饰符 按键别名 系统修饰键 鼠标按钮修饰符

按键别名

用如下修饰符来实现仅在按下相应按键时才触发鼠标或键盘事件的监听器。
.ctrl

这些修饰符会限制处理函数仅响应特定的鼠标按钮。
.left

.exact 修饰符允许我们控制由精确的系统修饰符组合触发的事件。

动态参数

也可以在指令参数中使用 JavaScript 表达式，方法是用方括号括起来

v-model

双向绑定表单元素（input,textarea,select…）的value值

修饰符 .lazy .number .trim
在默认情况下，v-model 在每次 input 事件触发后将输入框的值与数据进行同步 (除了上述输入法组织文字时)。你可以添加 lazy 修饰符，从而转为在 change 事件_之后_进行同步

v-if v-else-if v-else

控制绑定元素的有无

v-show

控制绑定元素的显示隐藏
v-show相当于在css中添加display:none,v-if则是删除/添加整个dom元素
如果元素频繁显示隐藏，就用v-show，在运行条件很少改变时，则使用v-if比较好

v-for

渲染出一个列表，可以生成多个绑定的元素
可以用v-for指令基于一个数组来渲染一个列表，也可以用v-for 来遍历一个对象的propert
Vue官方文档表示不推荐同时使用v-if和v-for，当v-if和v-for同时使用时，v-if具有更高的优先级
key 用于决定列表元素的唯一性，即代表这个元素
官方文档建议尽可能在使用v-for时提供key attribute

日期

创建日期对象的四种方法

注意：JavaScript 将日期存储为自 1970 年 1 月 1 日 00:00:00 UTC（协调世界时）以来的毫秒数。
零时间是 1970 年 1 月 1 日 00:00:00 UTC。
一天（24 小时）是 86 400 000 毫秒。

四种Javascript输入日期格式

  1. ISO日期:”2021-06-30” (国际标准)
  2. 短日期:”06/30/2021”或者”2021/06/30”
  3. 长日期:”Jun 30 2021”或者”30 Jun 2021”
  4. 完整日期:”Wednesday June 30 2021”
    其中ISO格式遵守JavaScript中的严格标准

JavaScript日期输出格式

默认输出全文本字符串格式

日期获取方法

  1. getDate() 以数值返回天（1-31）
  2. getDay() 以数值获取周名（0-6）
  3. getFullYear() 获取四位的年（YYYY）
  4. getHours() 获取小时（0-23）
  5. getMilliseconds() 获取毫秒（0-999）
  6. getMinutes() 获取分钟（0-59）
  7. getMonth() 获取月（0-11）
  8. getSeconds() 获取秒（0-59）
  9. getTime() 获取时间（从1970年1月1日至今）

日期设置方法

  1. setDate() 以数值（1-31）设置日
  2. setFullYears() 设置年
  3. setMonth() 设置月（0-11）
  4. setHours() 设置小时（0-23）
  5. setMinutes() 设置分钟（0-59）
  6. setSeconds() 设置秒（0-59）
  7. setMilliseconds() 设置毫秒（0-999）
  8. setTime() 设置时间（从1970年1月1日至今的毫秒数）

正则表达式基本内容

含义

正则表达式是用于匹配字符串中字符组合的模式，在JavaScript中，正则表达式也是对象。

作用

匹配特殊字符或有特殊搭配原则的字符的最佳选择

两种方法构建正则表达式

  1. 使用正则表达式字面量，其由包含在斜杠之间的模式组成

  2. RegExp对象的构造函数

相关

属性

  1. i 执行对大小写不敏感的匹配
  2. g 执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）
  3. m 执行多行匹配

表达式

  1. [abc] 查找方括号之间的任何字符
  2. [^abc] 查找任何不在方括号之间的字符
  3. [0-9] 查找任何从0-9的数字
  4. [a-z] 查找任何从小写a到小写z的字符
  5. [A-Z] 查找任何从大写A到大写Z的字符
  6. [A-z] 查找任何从大写A到小写z的字符
  7. [adgk] 查找给定集合内的任何字符
  8. [^adgk] 查找给定集合外的任何字符
  9. (red|blue|green) 查找任何指定的选项

模版字符串

含义

模版字符串也叫模版字面量，是允许嵌入表达式的字符串字面量，可以使用多行字符串和字符串插值功能

基础语法

  1. 最简单的用法看起来只是用反撇号（`)替换了单、双引号
  2. 想在字符串中使用反撇号，那么用反斜杠（\）将它转义即可，而在模版字面量中，不需要转义单、双引号
